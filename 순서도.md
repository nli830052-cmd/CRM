# 📊 SalesMind: 프로젝트 개발 순서도 (Roadmap)

이 문서는 AI 기반 영업 CRM **SalesMind**를 개발하기 위한 단계별 실행 계획을 담고 있습니다.

---

## 🛠️ Phase 1: 백엔드 기초 공사 (FastAPI + PostgreSQL)
- **목표**: 데이터 저장소와 API 서버의 뼈대를 만듭니다.
- [ ] FastAPI 프로젝트 초기화 및 가상환경 설정
- [ ] PostgreSQL 데이터베이스 연동 환경 구축
- [ ] 데이터 모델링 (SQLAlchemy): `User`, `Contact`, `Call`, `Message`, `AISummary`

## 🌐 Phase 2: 데이터 수집 API 개발
- **목표**: 안드로이드 앱에서 보내는 정보를 수신하여 DB에 저장합니다.
- [ ] 연락처 관리 API (`POST /contacts`, `GET /contacts`)
- [ ] 통화 및 문자 이력 적재 API (`POST /calls`, `POST /messages`)
- [ ] 특정 고객의 통합 히스토리 조회 API (`GET /timeline/{contact_id}`)

## 🧠 Phase 3: AI 분석 엔진 고도화 (OpenAI)
- **목표**: 축적된 대화 이력을 AI가 분석하여 요약 및 추천을 생성합니다.
- [ ] OpenAI API 연동 및 프롬프트 로직 설계
- [ ] 대화 요약(Summarizer) 기능 구현
- [ ] 다음 행동(Next Action) 추천 기능 구현
- [ ] 고객 상태(관심도) 분석 기능 구현

## 📱 Phase 4: 안드로이드 데이터 연동 및 UI
- **목표**: 실제 스마트폰 데이터를 서버로 전송하고 분석 결과를 시각화합니다.
- [ ] 통화 기록(`CallLog`) 및 문자(`SMS`) 퍼미션 획득
- [ ] 백그라운드 데이터 동기화 로직 구현 (Retrofit2 활용)
- [ ] **AI 인사이트 대시보드** UI 구현
- [ ] 통합 타임라인(문자+통화 믹스) UI 구현

## 🔔 Phase 5: 리마인드 및 고도화
- **목표**: 실사용성을 높이는 추가 기능을 구현합니다.
- [ ] AI 추천 일정 푸시 알림 연동
- [ ] 대화 내용 내 주요 키워드 추출 및 태그화
- [ ] 전체 영업 통계 대시보드 완성
